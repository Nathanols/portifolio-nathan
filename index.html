Habilidades

O JavaScript é uma linguagem de programação utilizada principalmente no desenvolvimento web. Ele é fundamental para criar sites interativos, dinâmicos e responsivos. O JavaScript é executado diretamente no navegador do usuário (lado do cliente), embora também possa ser usado no servidor, com o auxílio de ambientes como Node.js.

CSS (Cascading Style Sheets, ou Folhas de Estilo em Cascata) é uma linguagem de estilo usada para descrever a apresentação de documentos HTML ou XML. O CSS define como os elementos em uma página web devem ser exibidos — ou seja, ele controla o layout, as cores, a tipografia, as margens, os espaços, a disposição dos elementos, entre outros aspectos visuais.

HTML (HyperText Markup Language, ou Linguagem de Marcação de Hipertexto) é a linguagem padrão usada para criar e estruturar o conteúdo de páginas na web. Ele não é uma linguagem de programação, mas sim uma linguagem de marcação que define a estrutura básica de um site, incluindo textos, imagens, links, tabelas, listas e outros elementos.

A linguagem de programação C é uma linguagem de programação de baixo nível, desenvolvida por Dennis Ritchie no início da década de 1970, no Laboratório de Pesquisa de Computação da AT&T. Ela foi inicialmente criada para escrever o sistema operacional Unix, mas rapidamente se tornou uma das linguagens de programação mais populares e influentes devido à sua simplicidade, eficiência e portabilidade.

Java é uma linguagem de programação orientada a objetos e de alto nível, criada por James Gosling e desenvolvida inicialmente pela Sun Microsystems (agora parte da Oracle) na década de 1990. A linguagem foi projetada com o objetivo de ser portável, segura, robusta e de alto desempenho. O slogan do Java é "Write Once, Run Anywhere" (Escreva uma vez, execute em qualquer lugar), o que significa que o código Java pode ser executado em qualquer dispositivo que tenha uma Java Virtual Machine (JVM), sem a necessidade de recompilar.

Power BI é uma plataforma de análise de negócios desenvolvida pela Microsoft, que permite que usuários criem relatórios interativos e painéis de controle (dashboards) a partir de dados de várias fontes. Ele foi projetado para transformar dados brutos em informações significativas e visuais, ajudando empresas a tomar decisões baseadas em dados. A plataforma é amplamente utilizada por analistas de dados, profissionais de TI e executivos para explorar, visualizar e compartilhar informações de forma eficaz. O Power BI integra facilmente com diversas fontes de dados, como bancos de dados, planilhas Excel, serviços na nuvem e até mesmo fontes de dados web.

SQL (Structured Query Language) é uma linguagem de programação padronizada usada para gerenciar bancos de dados relacionais e realizar diversas operações em dados armazenados, como consultas, inserções, atualizações e exclusões. SQL é fundamental para trabalhar com bancos de dados e é amplamente utilizada por desenvolvedores, analistas de dados e administradores de banco de dados (DBAs) para manipulação de dados.
